# 离线图像转文字识别(OCR)

![Platform][platform_image]
[![API][min_api_image]][min_api_link]
[![License][license_image]][license_link]

## 特点
- 可以在离线的环境下，进行OCR识别
- 并且支持拍照或者图库选取待转换的图片
- 自动加载最新的图片(如果开启)
